To unlock all features and run the complete e-commerce platform:

Quick Setup Steps:

  1. Download Node.js: Go to https://nodejs.org/
  2. Install: Run the installer (choose LTS version)
  3. Restart: Restart your command prompt
  4. Install dependencies: Run npm install
  5. Setup database: Run npm run seed
  6. Start server: Run npm start
  7. Open: Go to http://localhost:3000
